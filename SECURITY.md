# Security Policy

## Supported Versions

No version is currently supported as this package is for experimentation.

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| 0.2.x   | :x:                |

## Reporting a Vulnerability

Please report vulnerabilities to the authors, either via email or the ESGF Slack.
