Security Policy
===============

Supported Versions
------------------

No version is currently supported as this package is for
experimentation.

.. list-table:: Supported Versions
   :widths: 25 50
   :header-rows: 1

   * - Version
     - Supported
   * - 0.3.1
     - In development
   * - 0.3.0
     - Accepting bug reports

Please report vulnerabilities to the authors, either via email or the
ESGF Slack.
